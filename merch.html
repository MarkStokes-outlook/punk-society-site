“Install the DLC” hoodie

Black heavyweight hoodie, neon circuit print, sleeve patch that reads “Patch Notes Pending”. QR code inside the hem links straight to the manifesto.

  • Colourway: matte black + electric cyan.
  • Slogan: “Society isn’t broken — it’s unfinished.”
  • Add-on: Patch Notes insert explaining the metaphor.

“Built Different” tee

Split graphic: Linux penguin silhouette vs glitching Windows logo with tagline “Linux brains in a Windows world.”

  • Organic cotton, tagless for sensory comfort.
  • Distressed print so it feels lived-in on day one.
  • Back print: “Ease is empathy.”

“Upgrade Required: Empathy” sticker pack

System alert pop-ups with buttons labelled “Install Update”, “Enable Sensory Mode”, and “Patch Cognitive Friction”.

  • 3 colourways (neon magenta, cyan, monochrome).
  • Matte finish so you can write notes on them.
  • Backer card doubles as a mini-friction checklist.

Production notes

  • Palette: matte black, electric cyan, punchy magenta, chalk white.
  • Typography: monospace for code vibes, bold sans for slogans.
  • Fabrics: organic cotton, recycled blends, tagless & sensory-friendly.
  • Packaging: “Patch Notes” insert with QR links to manifesto + story intake.
